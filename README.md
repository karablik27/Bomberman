# Bomberman - iOS Multiplayer Game
## Описание проекта

**Bomberman** — это iOS-игра в жанре аркадного многопользовательского экшена, вдохновлённая классическими играми серии Bomberman. 

Проект ориентирован на совместную игру нескольких пользователей в реальном времени с акцентом на динамичный геймплей, кастомизацию и взаимодействие между игроками.

Игровой процесс строится вокруг сражений в лабиринте, состоящем из:
- **неразрушаемых стен**, формирующих структуру уровня;
- **разрушаемых блоков**, которые могут быть уничтожены с помощью бомб.

Каждый игрок управляет своим персонажем, перемещается по полю и устанавливает бомбы. Бомба взрывается через фиксированное время и поражает клетки по горизонтали и вертикали, что позволяет:
- разрушать блоки;
- уничтожать противников;
- открывать путь к новым зонам карты.

### Цель игры

Основная цель каждого раунда — остаться **единственным выжившим игроком** на игровом поле либо победить по истечении времени, если противники были устранены ранее. Раунд завершается автоматически после выполнения одного из условий победы.

### Ключевые особенности проекта

- **Многопользовательский режим**  
  Поддержка лобби, в котором несколько игроков могут подключаться к одной игровой сессии.

- **Выбор персонажа и имени**  
  Перед началом игры пользователь вводит имя и выбирает визуальный скин персонажа.

- **Кастомный пользовательский интерфейс**  
  Интерфейс игры выполнен в едином стиле с анимированными фонами и интерактивными элементами.

- **Встроенный чат**  
  Игроки могут общаться между собой в лобби до начала матча.

- **Настройки музыки и звуков**  
  Возможность включения/отключения музыки и звуковых эффектов, а также установки пользовательской музыки.

- **Адаптация под iOS**  
  Проект разработан с учётом iOS-гайдлайнов и корректно работает на различных размерах экранов.

### Назначение проекта

Проект был разработан в рамках **хакатона №3** как демонстрация:
- навыков разработки iOS-приложений;
- проектирования пользовательского интерфейса;
- реализации логики многопользовательской игры;
- командной работы над программным продуктом.

Игра может быть расширена за счёт добавления новых карт, игровых режимов, бонусов и улучшенной сетевой логики.
